# Animagine XL 3.1 参数与提示词中文指南

> 基于官方文档整理 | 适用于 FaceOrbit 高级模式

## 一、模型简介

Animagine XL 3.1 是 Animagine XL V3 系列的最新版本，基于 Stable Diffusion XL 构建，专为生成高质量动漫风格图像而设计。它增强了手部解剖学、概念理解和提示词解析能力。

**开发方**：Cagliostro Research Lab  
**模型类型**：基于扩散的文本生成图像模型  
**许可证**：CreativeML Open RAIL++-M License

---

## 二、推荐参数设置（官方最佳实践）

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| **采样器 (Sampler)** | `euler_ancestral` (Euler a) | 官方强烈推荐 |
| **采样步数 (Steps)** | `28 - 30` | **低于 30 步效果最好** |
| **CFG Scale (引导强度)** | `5 - 7` | 越低模型越自由，越高越遵循提示词 |
| **分辨率** | 见下表 | 支持多种宽高比 |

### 支持的分辨率

| 尺寸 | 宽高比 |
|------|--------|
| 1024 x 1024 | 1:1 方形 |
| 1152 x 896 | 9:7 |
| 896 x 1152 | 7:9 |
| 1216 x 832 | 19:13 |
| **832 x 1216** | **13:19（推荐竖版）** |
| 1344 x 768 | 7:4 横向 |
| 768 x 1344 | 4:7 纵向 |
| 1536 x 640 | 12:5 横向 |
| 640 x 1536 | 5:12 纵向 |

## 三、提示词标签系统

Animagine XL 3.1 针对 **Danbooru 风格标签** 优化，而非自然语言。

### 📌 标签顺序模板（官方推荐）

```
1girl/1boy, 角色名称, 作品来源, 其他描述（顺序不限）
```

**示例**：
```
1girl, souryuu asuka langley, neon genesis evangelion, solo, upper body, smile, looking at viewer, outdoors, night
```

## 四、特殊标签（质量/评级/年代/美学）

### 1. 质量修饰符（Quality Modifiers）

| 标签 | 分数标准 | 使用建议 |
|------|----------|----------|
| `masterpiece` | > 95% | 追求最高质量时使用 |
| `best quality` | > 85% - 95% | **日常推荐** |
| `great quality` | > 75% - 85% | 快速生成 |
| `good quality` | > 50% - 75% | 测试时使用 |
| `normal quality` | > 25% - 50% | 不推荐 |
| `low quality` | > 10% - 25% | 避免使用 |
| `worst quality` | ≤ 10% | 避免使用 |

**官方推荐开场组合**：
```
masterpiece, best quality, very aesthetic, absurdres
```

### 2. 评级修饰符（Rating Modifiers）

| 标签 | 含义 | 说明 |
|------|------|------|
| `safe` | 全年龄 | 一般内容 |
| `sensitive` | 敏感 | 轻度敏感内容 |
| `nsfw` | 成人 | 成人内容 |
| `explicit` | 明确 | 明确成人内容 |

### 3. 年代修饰符（Year Modifiers）

| 标签 | 年份范围 | 风格特点 |
|------|----------|----------|
| `newest` | 2021 - 2024 | 现代动漫风格 |
| `recent` | 2018 - 2020 | 近期风格 |
| `mid` | 2015 - 2017 | 中期风格 |
| `early` | 2011 - 2014 | 早期风格 |
| `oldest` | 2005 - 2010 | 复古动漫风格 |

### 4. 美学修饰符（Aesthetic Tags）

基于 ViT 模型评估的视觉美感：

| 标签 | 分数范围 | 含义 |
|------|----------|------|
| `very aesthetic` | > 0.71 | 极具美感 |
| `aesthetic` | > 0.45 - 0.71 | 有美感 |
| `displeasing` | > 0.27 - 0.45 | 不美观 |
| `very displeasing` | ≤ 0.27 | 非常不美观 |

## 七、快速调参参考

| 目标 | 参数调整建议 |
|------|-------------|
| **更像上传照片** | 提高 `ip_weight` (0.85-0.95)，降低 `denoise` (0.65-0.7) |
| **更自由发挥** | 降低 `ip_weight` (0.7-0.8)，提高 `denoise` (0.75-0.8) |
| **更遵循角度提示词** | 提高 `cn_strength` (0.45-0.6) |
| **更鲜艳色彩** | 提高 `correction_strength` (0.7-0.8) |
| **提高清晰度** | 增加 `steps` (35-40)，使用 `euler_ancestral` |

## 八、已知局限

1. **仅限动漫风格**：不适合生成真实照片
2. **需要详细提示词**：简短提示词效果不佳
3. **标签格式**：使用 Danbooru 标签而非自然语言
4. **手部渲染**：虽已改进，但偶尔仍有瑕疵
5. **NSFW 内容**：可能生成未提示的成人内容

---
